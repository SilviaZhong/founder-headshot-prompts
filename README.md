# Founder Headshot Prompts

Turn a plain ID photo into a realistic, startup-founder-style professional portrait.

Designed for:
- startup competitions
- demo day / pitch decks
- AI business presentations
- team intro slides
- About pages

## What’s inside
- Stable prompt for women
- Stable prompt for men
- Universal editable template
- Agent Skill version for Codex-compatible tools

## Output style
- realistic
- clean
- minimal
- founder / product-builder vibe
- not overly corporate
- not obviously AI-generated

## Quick start
1. Upload your original ID photo
2. Copy one of the prompts from `prompts/`
3. Paste it into ChatGPT image generation or a compatible image model
4. Adjust age / hairstyle / outfit if needed

## Files
- `prompts/female-stable-prompt.md`
- `prompts/male-stable-prompt.md`
