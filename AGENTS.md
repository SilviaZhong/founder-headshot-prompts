# Repository expectations

## Purpose
This repository stores reusable prompts and an instruction-only skill for generating realistic startup-founder professional portraits from original ID photos.

## Editing rules
- Keep prompts concise and reusable
- Prioritize realism over stylization
- Avoid corporate stock-photo language
- Avoid over-beautification and obvious AI artifacts
- Keep examples privacy-safe
- When updating prompt files, also update README if usage changes

## Naming
- Use simple, direct filenames
- Separate female, male, and universal templates
